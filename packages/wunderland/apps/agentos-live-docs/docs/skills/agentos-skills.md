`@framers/agentos-skills` is the **content package** — it ships 69 curated SKILL.md files and a `registry.json` index. It contains no runtime code.

```bash
npm install @framers/agentos-skills
```

For the catalog SDK (query helpers, factories, typed catalog), see [`@framers/agentos-skills-registry`](./agentos-skills-registry).

For the skills engine (SkillLoader, SkillRegistry, path utils), use `@framers/agentos/skills`.

For the tool extension (skills_list, skills_read, etc.), use `@framers/agentos-ext-skills`.
